# 하이리온 시연 시나리오

**총 시간:** 약 4분 | **구성:** 등장 → 인사 → 대화+집기 → 보행 → 마무리

---

## Phase 1 — 등장 (0:00 ~ 0:30)

> 초기 상태: 무대 한쪽에서 IDLE 대기

| 시각 | 트리거 | 동작 |
|------|--------|------|
| 0:00 | 진행자: "하이리온, 나와봐!" | BHL 보행으로 테이블 앞까지 이동 (5~7보)<br>테이블 앞 지정 위치에서 정지 |
| 0:20 | 정지 | WALKING → IDLE 전환<br>잠깐 멈추며 관객 바라보는 포즈 (양팔 자연스럽게 내림) |

## 비상 시나리오 B (보행 미달 시)

- **Phase 1 (등장)** 생략, 받침대 고정 상태로 시작
- **Phase 3 (집기)** 사이 간격을 늘려 진행자와 자유 대화 추가 → 4분 채움
- **Phase 4 (보행)** 대신 양팔 동시 동작 (precoded 박수 또는 양팔 스트레칭)으로 대체
